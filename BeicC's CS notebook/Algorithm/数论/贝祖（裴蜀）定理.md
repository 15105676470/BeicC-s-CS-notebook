看这个定理的时候可以复习一下拓展欧几里得定理，感觉这俩关系好密切

贝祖定理内容：1：原始形式：ax + by = 1 有解的充要条件是a 和 b 互质

​							2：一般形式：ax + by = c 有解的充要条件是c | gcd（a，b）

证明：对于一般形式，设s  = gcd（a，b），那么有 ax | s  && by | s ，所以c | s，证毕。

拓展成n元关系：a1x1 + a2x2 + ..... + anxn = s 的充要条件是s = K * gcd（a1,a2,....an）；
